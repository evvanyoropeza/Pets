# 🐾 Pets API - Technical Test

## 📌 Descripción
API REST que consume una API externa para gestión de mascotas.

## 🛠️ Tecnologías
- Java
- Spring Boot
- OpenFeign
- Gradle

## 🚀 Endpoints

### GET /api/pet/{id}
Obtiene una mascota

### POST /api/pet
Crea una mascota

## ▶️ Ejecutar

./gradlew bootRun

## 📌 Notas
- No usa base de datos
- Arquitectura por capas
- Manejo de errores global
